### 스프링 시큐리티 의존성 추가

- 스프링 시큐리티 의존성 추가시 일어나는 일들
  - 서버 기동하면 스프링 시큐리티 초기화 작업 및 보안 설정이 이루어짐
  - 별도 구현없이 기본적인 웹 보안 기능이 세팅되어 작동함
    - 모든 요청은 인증이 되어야 자원에 접근 가능해짐
    - 인증 방식은 폼 로그인 방식과 httpBasic 로그인 방식을 제공
    - 기본 로그인 페이지 제공
